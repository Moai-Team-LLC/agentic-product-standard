# Support

Thanks for using the Agentic Product Standard. Here's the fastest way to get help.

## Where to go

| You want to… | Go to |
| --- | --- |
| **Ask a question** ("how do I apply X to my agent?") | [GitHub Discussions](https://github.com/Moai-Team-LLC/agentic-product-standard/discussions) |
| **Report an error** in the standard or a skill | [Open an issue](https://github.com/Moai-Team-LLC/agentic-product-standard/issues/new/choose) |
| **Propose a change** (new exemplar, framework update, translation) | [Open a PR](CONTRIBUTING.md) |
| **Report a security/safety problem** | [Security policy](SECURITY.md) — private advisory, not a public issue |

## Before you ask

- Skim [`STANDARD.md`](STANDARD.md) and the relevant [`skills/`](skills/agentic-product-architect) sub-skill — most "how do I…" questions are answered there.
- Search [existing issues](https://github.com/Moai-Team-LLC/agentic-product-standard/issues?q=) and [discussions](https://github.com/Moai-Team-LLC/agentic-product-standard/discussions).

## What this project does and doesn't cover

This repo is **vendor-neutral architectural guidance**, not a support desk for any specific
framework or model. Questions about *applying the standard* are welcome; questions about a
third-party tool's bugs belong in that tool's tracker.

Maintained by [Moai Team LLC](https://moaiteam.com). Best-effort, no SLA.
